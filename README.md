# 🚚 Lowcountry Labor Co. - Small Business Website

A clean, mobile-friendly website built for a local service business — **Bryce's Services** — offering pressure washing, moving, junk removal, and other labor-based services in the Bluffton and Hilton Head, SC area.

🛠 Built with **React + TailwindCSS + DaisyUI**, and deployed via **Vercel**.

---

## 🌐 Live Site

🔗 [lowcountrylaborco.org](http://lowcountrylaborco.org)

---

## 📋 Features

- ⚡ Fast-loading and responsive design
- 🧽 Emphasis on pressure washing (featured service)
- 📷 Before/after image showcase
- ✅ Services grid with animations
- 📝 Contact form with budget, service selection, and location fields
- 📱 Mobile-first design with interactive transitions

---

## 🧪 Tech Stack

**Frontend**
- React (Vite)
- TailwindCSS
- DaisyUI
- Lucide Icons
- Framer Motion (animations)
- Deployed via **Vercel**
